Introduzione a JavaScript e gli Eventi

JavaScript (JS) è un linguaggio di programmazione usato nei browser per rendere le pagine web interattive. Ad esempio, puoi far succedere qualcosa quando l'utente clicca un bottone, scrive un testo o muove il mouse.

Gli eventi sono "azioni" che accadono sulla pagina (come un clic o una pressione di tasto). JS può "ascoltare" questi eventi con la funzione addEventListener, e rispondere quando accadono.
