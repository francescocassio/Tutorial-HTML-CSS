Spiegazione dettagliata di flex-basis

flex-basis definisce la dimensione iniziale di un elemento in un contenitore flessibile. Funziona come una "larghezza minima" prima che intervengano flex-grow o flex-shrink.

1️⃣ flex-basis con valore uguale

Tutti gli elementi iniziano con una larghezza di 150px.

Box 2
Box 3

2️⃣ flex-basis con valori diversi

Ogni elemento ha una dimensione iniziale differente.

Box 1 (100px)
Box 2 (200px)
Box 3 (300px)

3️⃣ flex-basis con flex-grow

Il primo box cresce normalmente, il secondo cresce il doppio.

Box 1 (grow 1)
Box 2 (grow 2)

4️⃣ flex-basis con flex-shrink

Il primo box non si riduce mai, il secondo sì.
