Spiegazione dettagliata di flex-shrink

flex-shrink controlla quanto un elemento può ridursi quando lo spazio è insufficiente. Se il valore è 0, l'elemento non si ridurrà mai. Se è maggiore di 1, si ridurrà più velocemente degli altri.

1️⃣ flex-shrink uguale per tutti

Tutti gli elementi hanno flex-shrink: 1, quindi si riducono in modo equo se lo spazio non basta.

Box 3

2️⃣ flex-shrink con valori diversi

Il primo box non si riduce (flex-shrink: 0), il secondo si riduce normalmente e il terzo si riduce tre volte più degli altri.

Box 2 (shrink 1)
Box 3 (shrink 3)

3️⃣ flex-shrink con flex-basis

Il primo box può ridursi, il secondo ha flex-shrink: 0 quindi mantiene sempre 250px.

Box 2 (no shrink)
